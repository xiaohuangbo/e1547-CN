## 特性

- 跨平台 (Android,~~iOS~~)
- 浏览帖子和图集
- 编辑帖子
- 评论帖子
- 下载图片
- 收藏、赞成和反对帖子
- 关注标签并接收通知
- 本地黑名单
- DText 解析
- 视频支持
- 多账户登录
- 多种应用主题

## 安装

### 在 Android 上安装

1. 下载 [最新的 APK](https://github.com/xiaohuangbo/e1547-CN/releases/latest)
2. 在你的 Android 设备上使用文件管理器打开它
3. 点击安装

## 编译

你可以从源代码编译该应用：

1. 安装 [Flutter](https://flutter.dev/docs/get-started/install)
2. 克隆此 github 仓库
3. 运行 `flutter build <file>`，其中 `<file>` 是 `apk` 或 `ipa`
